## ✅ Fonctionnalités

- 📅 **Plan d'étude — 31 jours** : table des matières officielle du livre (chapitres exacts)
- ✏️ **Exercices pratiques** : 2 questions par jour avec correction détaillée
- 🧪 **Labs Cisco** : 31 labs Packet Tracer / CLI IOS avec topologies, commandes et vérifications
- 🎯 **Quiz CCNA** : 20 questions aléatoires avec explications (alignées sur les chapitres)
- 📋 **Aide-mémoire** : subnetting, distances administratives, OSI, ports, OSPF, STP, WPA...
- 💾 **Progression sauvegardée** automatiquement (localStorage)
- ⚡ **Fonctionne hors ligne** : un seul fichier HTML, aucune installation requise

---

## 🚀 Installation et utilisation

```bash
# 1. Cloner le dépôt
git clone https://github.com/TON-USERNAME/ccna-31days-app.git

# 2. Ouvrir le fichier dans votre navigateur
# (Double-clic sur le fichier)
open CCNA_31Days_v4.html
```

> **Aucune dépendance, aucun serveur, aucune connexion Internet requise.**
